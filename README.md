# solokhin.dev

This is my personal website, built with [Astro](https://astro.build) and deployed on [Cloudflare Workers](https://workers.cloudflare.com).

## Commands

| Command                  | What it does                                            |
| :----------------------- | :------------------------------------------------------ |
| `npm run dev`            | Starts the local dev server at `localhost:4321`         |
| `npm run build`          | Builds the production site to `./dist/`                 |
| `npm run preview`        | Previews the build locally before deploying             |
| `npm run astro ...`      | Runs Astro CLI commands like `astro add`, `astro check` |
| `npm run generate-types` | Generates TypeScript types for Cloudflare bindings      |
| `npm run deploy`         | Deploys the production site to Cloudflare               |
